VOX Conference Izlozes Lapa
==============================

1. Augšupielādē visus failus GitHub repozitorijā (piemēram, `vox-conference-draw`)
2. GitHub repozitorijā ej uz Settings > Pages un izvēlies branch: `main`, folder: `/root`
3. URL būs `https://<tavs-lietotājvārds>.github.io/vox-conference-draw/`
4. Vari ģenerēt QR kodu, kas ved uz šo saiti

Visi dalībnieki tiek saglabāti pārlūkā. Lapa neizmanto serveri vai datubāzi.
